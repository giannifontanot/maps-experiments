# maps-experiments

Simple front-end app that displays a Google Map with nearby businesses and a
clickable list of the closest results.

## Getting started

1. Add your Google Maps JavaScript API key in `index.html`.
2. Start a local server:

```bash
python -m http.server 8000
```

3. Visit `http://localhost:8000` and click **Use my location**.

> Note: The Places API is required for nearby business results.
